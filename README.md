# Flappy Bird in Python (15-Minute Build)

A simple Flappy Bird clone built in Python using Pygame.

This project was built as part of my **"How YOU can code X in 15 minutes"** series on YouTube.

## Features
- Gravity-based movement
- Jump mechanics (spacebar / click)
- Random pipe generation
- Collision detection
- Score tracking
- Game over system

---

## How It Works
The game is built using a simple game loop:

1. Apply gravity to the bird
2. Allow player input (jump)
3. Move pipes across the screen
4. Detect collisions
5. Track score when passing pipes
